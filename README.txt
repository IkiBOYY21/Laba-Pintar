1. Import file SQL Anda (db_labapintar.sql) ke phpMyAdmin.
2. Jalankan perintah ALTER TABLE untuk menambahkan AUTO_INCREMENT (lihat instruksi di awal).
3. Letakkan folder proyek labapintar di C:\xampp\htdocs\labapintar
4. Buka config.php, sesuaikan DB credentials dan BASE_URL.
5. Jalankan Apache & MySQL di XAMPP.
6. Akses http://localhost/labapintar/
7. Register user baru -> login -> mulai gunakan.
